LIBA v2.0
=========
Versi ini mengandungi:
- index.html (UI LIBA v2.0)
- liba.css (gaya)
- app.js (fungsi asas: mula/tamat baca, transkrip automatik)
- manifest.json (untuk PWA / Add to Home Screen)
- sw.js (service worker untuk offline)
Untuk guna:
1. Letak semua fail dalam satu folder.
2. Sediakan folder icons/ dan masukkan logo-skbt.png, logo-ts25.png, logo-liba.png.
3. Buka index.html dalam Microsoft Edge / Chrome.
4. Untuk fungsi PWA penuh, host di GitHub Pages / Firebase.
